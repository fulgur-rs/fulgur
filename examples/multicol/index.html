3 columns, narrow gap

新聞や雑誌で馴染みのある段組みレイアウトは、長い本文を読みやすい幅に分割し、ページ内の情報密度を高めるために使われる。ブラウザは column-count や column-width の指定に従い、要素内のコンテンツをバランスよく複数のカラムへ流し込む。fulgur もこの動きを再現すべく、Parley のレイアウト結果を列幅で再ブレイクしてから PDF へ描画する。

column-width driven

When column-width is specified instead of column-count, the browser derives the column count from the container width using floor((content + gap) / (width + gap)). The declared width acts as a target; actual columns are slightly wider so they tile perfectly. Narrow viewports gracefully fall back to a single column.
